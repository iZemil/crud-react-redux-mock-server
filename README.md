# Simple CRUD with react, redux, mock-server

```
yarn server
```
```
yarn start
```

### Модели

* Department (name: string)
* Employee (firstName: string, lastName: string, departmentId:number) 

### Интерфейс (Левое меню)

* Departments
* Employees

### Контент

* Таблица редактирования выбранной сущности
